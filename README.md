# group-chat-and-manage-data

## Prequisite
1. NodeJS v18.13.0
2. PostgresSQL 10.23

## Install knex globbaly
`npm install knex -g`

## Migrate the Database and insert seed data
`knex migrate:latest; knex seed:run;`

## Install all dependencies
`npm install`

## Run the server
`npm start`

## Testing application by running test script
`npm run test`
